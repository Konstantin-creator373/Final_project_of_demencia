# Dementia Risk Tracker

# Что это такое?
Этот проект это анонимное веб-приложение опросник, которое запрашивает информацию о состоянии пользователя
и анализирует полученные результаты, предупреждая пользователя при наличии опасности ранней степени деменции.

Так же присутствует админ страница для просмотра результатов.

# Целевая аудитория

Пенсионеры и люди среднего возраста.

# Когда будет нужен проект?

Данный проект будет востребован круглогодно.
